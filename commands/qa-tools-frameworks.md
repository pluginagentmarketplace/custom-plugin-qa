# QA Tools & Frameworks Guide

Explore and select the right tools for your testing needs.

## Test Automation

| Tool | Best For | Learning Curve |
|------|----------|-----------------|
| Selenium | Web apps | Medium |
| Cypress | Modern web | Easy |
| Playwright | Multi-browser | Easy |
| Appium | Mobile apps | Medium |

## API Testing

- Postman (Easy, GUI)
- REST Assured (Java)
- Supertest (Node.js)
- Insomnia (Modern REST)

## Performance Testing

- JMeter (Comprehensive)
- k6 (Modern, scriptable)
- Locust (Python-based)
- Gatling (High load)

## CI/CD

- GitHub Actions (GitHub native)
- Jenkins (Self-hosted)
- GitLab CI (GitLab native)
- CircleCI (Cloud)

## Selection Criteria

✅ Team expertise
✅ Project needs
✅ Budget
✅ Community support
✅ Integration capabilities

Use `/qa-projects` to practice!
