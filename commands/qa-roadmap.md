# QA Engineer Roadmap

Complete learning path from beginner to expert QA Engineer.

## Phase 1: Foundations (4-6 weeks)

**Topics**
- QA fundamentals & methodologies
- Test types & approaches
- Test case design
- Bug reporting

**Projects**: 1-2
**Time**: 40-60 hours

## Phase 2: Manual Testing (2-4 weeks)

**Topics**
- Manual test execution
- Exploratory testing
- Test documentation
- Regression testing

**Projects**: 3-5
**Time**: 20-40 hours

## Phase 3: Test Automation (6-8 weeks)

**Topics**
- Selenium WebDriver
- Cypress & Playwright
- Mobile testing (Appium)
- Test frameworks

**Projects**: 6-10
**Time**: 60-80 hours

## Phase 4: Advanced Testing (6-8 weeks)

**Topics**
- API testing (REST & GraphQL)
- Performance testing (JMeter, k6)
- Security testing (OWASP)
- CI/CD integration

**Projects**: 11-18
**Time**: 60-80 hours

## Phase 5: Expert (4-6 weeks)

**Topics**
- Advanced frameworks
- Metrics & reporting
- Team leadership
- Process improvement

**Projects**: 19-25
**Time**: 40-60 hours

## Total: 6-12 months, 220-320 hours

Use `/qa-strategy` to plan your path!
