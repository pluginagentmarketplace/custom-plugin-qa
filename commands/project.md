# Projects: Hands-On Full Stack Learning

Explore and start hands-on projects to apply your Full Stack Development knowledge. Projects are categorized by skill level and learning area.

## ðŸŸ¢ Beginner Projects (Weeks 1-4)

### Frontend Projects

**1. Personal Portfolio Website**
- **Topics**: HTML, CSS, Responsive Design
- **Duration**: 1 week
- **Description**: Build a simple portfolio showcasing your projects
- **Requirements**:
  - Responsive design (mobile, tablet, desktop)
  - About, projects, contact sections
  - Clean, modern styling
  - GitHub link included
- **Skills Learned**: HTML5, CSS3, responsive design

**2. Interactive Todo App (Vanilla JS)**
- **Topics**: JavaScript DOM, Events, Local Storage
- **Duration**: 1 week
- **Description**: Build a todo list app with vanilla JavaScript
- **Requirements**:
  - Add, complete, delete todos
  - Local storage persistence
  - Filter by status (all, active, completed)
  - Clean UI
- **Skills Learned**: DOM manipulation, events, storage

**3. Weather App**
- **Topics**: Async JavaScript, APIs, DOM manipulation
- **Duration**: 1 week
- **Description**: Fetch weather data and display it
- **Requirements**:
  - Search by city
  - Display current weather
  - Show forecast
  - Use public API (OpenWeatherMap)
- **Skills Learned**: APIs, promises, async/await

### Backend Projects

**4. Simple Express Server**
- **Topics**: Node.js, Express basics, Routing
- **Duration**: 1 week
- **Description**: Create a basic API server
- **Requirements**:
  - Multiple routes
  - JSON responses
  - Error handling
  - Logging
- **Skills Learned**: Express, HTTP, routing

**5. REST API with JSON**
- **Topics**: Express, RESTful design, data storage
- **Duration**: 1-2 weeks
- **Description**: Build a simple REST API
- **Requirements**:
  - CRUD operations
  - JSON data file (or in-memory)
  - Proper HTTP methods
  - Status codes
- **Skills Learned**: REST principles, API design

### Full Stack Projects

**6. Blog Platform (Simple)**
- **Topics**: HTML, CSS, JS, Node.js, File system
- **Duration**: 2 weeks
- **Description**: Create a simple blog
- **Requirements**:
  - Create, read, delete posts
  - Simple markdown support
  - List posts
  - Archive functionality
- **Skills Learned**: Full stack basics, file system
