# QA Learning Path

Choose your personalized learning path based on your experience level.

## Beginner (4-6 weeks)

Start here if you're new to QA!

**Topics**
- QA fundamentals
- Testing types
- QA mindset
- Test planning basics
- Manual testing

**Projects**: 1-8

## Intermediate (6-8 weeks)

Have basic QA knowledge? Level up!

**Topics**
- Test design techniques
- Test automation frameworks
- API testing
- Performance testing
- CI/CD basics

**Projects**: 9-18

## Advanced (8-10 weeks)

Ready for expert level?

**Topics**
- Advanced automation
- Security testing
- QA metrics & reporting
- Team leadership
- Process improvement

**Projects**: 19-30

## Expert (Ongoing)

Continue mastering QA!

**Topics**
- Advanced architecture
- Innovation & optimization
- Team building
- Strategic quality

Use `/qa-roadmap` for detailed timeline!
