# Test Coverage & Risk Analysis

## Coverage Types

### Code Coverage
- Line coverage
- Branch coverage
- Function coverage
- Statement coverage

### Feature Coverage
- Feature coverage %
- User scenario coverage
- Business logic coverage

### Risk Coverage
- High-risk areas
- Priority testing
- Critical paths
- Customer impact

## Risk-Based Testing

1. **Identify Risks** - What could go wrong
2. **Assess Risk** - Probability and impact
3. **Prioritize** - Focus on high risks
4. **Test** - Comprehensive testing
5. **Monitor** - Defect tracking

## Targets

✅ Code coverage: 80%+
✅ Critical paths: 100%
✅ Risk coverage: 95%+
