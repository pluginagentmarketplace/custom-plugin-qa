# Performance Profiling & Optimization

## Profiling Tools

### Browser
- Chrome DevTools
- Firefox Developer Tools
- Network tab

### Application
- Java: JProfiler, YourKit
- Python: cProfile, profile
- Node.js: Node Inspector

✅ Response time
