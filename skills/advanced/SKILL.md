# Security Testing & OWASP

## OWASP Top 10

1. **Injection**: SQL injection, command injection
2. **Broken Authentication**: Weak passwords, session issues
3. **Sensitive Data Exposure**: Unencrypted data
4. **XML External Entities (XXE)**
5. **Broken Access Control**: Privilege escalation
6. **Security Misconfiguration**: Default configs
7. **Cross-Site Scripting (XSS)**: DOM injection
8. **Insecure Deserialization**
9. **Using Components with Known Vulnerabilities**
10. **Insufficient Logging & Monitoring**
