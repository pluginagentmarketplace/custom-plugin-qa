## Key Metrics

### Coverage
- Code coverage (80%+ target)
- Feature coverage
- Test case coverage

### Defects
- Defect density
- Escape rate
- Severity distribution
- MTTR (mean time to repair)

### Process
- Test execution rate
- Pass/fail ratio
- Automation %
- Team velocity
